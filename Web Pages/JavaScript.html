Introduction

Often abbreviated as JS, JavaScript is not actually a programming language like Java and for that matter it was not even developed by the same people. It is in actuality a scripting language or a "client-side" programming language. The code is utilised much like a programming language however, it is read, interpreted and executed by the web browser in order to control the behaviour of different elements. It is used in conjunction to HTML and CSS in order to create fully functional websites.
