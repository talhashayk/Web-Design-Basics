Introduction

Developed by Tim Berners-Lee in 1990, HTML is the standard mark-up language that is used when creating a new website. The acronym HTML stands for:

  • H - Hyper
  • T - Text
  • M - Markup
  • L - Language

Its implementation is quite basic as it is used to describe the general structure of Web pages. It utilises elements that are represented by tags as the building blocks of the site. There are many versions of HTML and the latest version of it happens to be HTML5 which was released in the year 2012.

Basics

HTML code for a webpage is mainly comprised of 2 major sections. These sections are the head and the body of the code. The head holds the link to any relating style sheet as well as the title and character set of the web page. The body holds the actual code that constructs the web page and is thus the main bulk of the code.

Tags

Tags are utilised to indicate the start and finish of a HTML element. They come in many variations and are thus, incredibly useful. There are tags to indicate the placement of an image, the location of a link, the start of a paragraph, many forms of headings... Evidently, this shows that there is a vast amount of tags and some of the basic tags will be listed below as well as how they work.

Validation

Validation methods are used in order to ensure that the maintenance and modification of code is possible without any serious issues. As well as this it also is used to ensure the developer does not utilise any deprecated code within the creation of their web page.

A commonly used validator tends to be the site W3C and this site not only informs the user of warnings and errors but it also provides descriptions of what they are thus, ensuring that their removal is done correctly and doesn't cause any further errors.
